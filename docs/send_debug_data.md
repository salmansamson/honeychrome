# Data and Privacy
We do not collect or process any personal data in Honeychrome.

There is a "Send Debug Data" setting to choose whether to send debug information to the developers if Honeychrome produces errors or crashes. No personal information or user data is included. If this setting is on, the version number of Honeychrome and name of the operating system is also sent.

## How to opt-out:
We believe in user choice. You can disable debug data by unchecking "Send Debug Data" 
- on splash screen on first start up
- subsequently in menu Edit > App Configuration 

## What we NEVER collect:
- Personally Identifiable Information (PII): We do not collect your name, email, IP address, or local filenames.
- User Data: We do not collect the contents of the files you process or your inputs.
- Hardware IDs: We do not track unique hardware serial numbers.

Honeychrome uses [sentry.io](https://sentry.io) for collecting and processing debug information.
